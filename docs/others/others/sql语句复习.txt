14.相等联接
	传统方法：select title,name from books,publisher where books.pubid = publisher.pubid;
	JOIN方法：（1）NATURAL JOIN关键字
			在包括一个同等命名和定义的字段的两个表之间自动创建联接
			select title,pubid,name from publisher NATURAL JOIN books;
		  （2）USING关键字
			根据在两个表中具有相同名称和定义的列创建联接
			select title,pubid,name from books JOIN publisher USING (pubid);
		  （3）JOIN...ON关键字
			当USING子句中联接的表没有没有包括一个同等命名和定义的字段时，你必须添加一个
			where子句或者向JOIN关键字添加ON子句，以便指定这两个表之间的关系
			select title,name from books b JOIN publisher p ON b.pubid=p.pubid;
	随JOIN关键字使用USING和ON子句，期间存在两个主要的区别：
	a.“只有”当联接的表包括一个具有相同名称的公共列时才能使用USING子句，这不是ON子句所必须的
	b.条件是在ON子句中指定的，这在USING子句中是不允许的。USING子句只能包含公共列的名称。

17.集合运算符
	UNION――返回结合的SELECT语句的结果。删除重复的记录
	UNION ALL――返回结合的SELECT语句的结果。不删除重复的记录
	INTERSECT――只返回两个SELECT语句的结果中都包括的行
	MINUS――删除第一个查询中也存在的第二个查询的结果，只显示由第一个查询返回的行
